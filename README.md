## 管理Kubernetes资源 

### 通过此平台可以实现如下功能可以操作
* namespace 
* deployment
* service
* ingress
* pod (webshell)
* cronjob
* node 节点管理
* 权限根据namespace分配
* 日志审计

***

### 部署

前端: [front](https://github.com/Arnold617/k8s-manage/tree/master/front)
后端: [backend](https://github.com/Arnold617/k8s-manage/tree/master/front)
